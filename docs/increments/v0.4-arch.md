## الجديد
```
lib/arch/
  roof.ts         — أسقف بارامترية: جملون/هرمي/مائل/مسطح (مسقط + مجسّم)
  stairs.ts       — درج بارامتري: حساب القائمة/النائمة/الزاوية + درجات 3D

```
