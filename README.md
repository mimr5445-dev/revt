# ◣ Revt — مصمّم المباني السحابي (BIM)

**Revt** تطبيق عربي احترافي لبناء النماذج المعمارية (BIM) يعمل بالكامل في المتصفح — مستوحى من Revit لكن خفيف وسحابي ومجاني ويعمل على الهاتف. ترسم المسقط، تشاهده ثلاثي الأبعاد فورًا، وتحصل على جدول الكميات والتكاليف من **معالج سحابي**.

## ✨ المميزات

- **محرّر مسقط 2D**: ارسم الجدران بالنقر، أضف أرضيات وأعمدة وأبواب ونوافذ، مع شبكة وتقريب (snap).
- **عرض ثلاثي الأبعاد فوري** (Three.js) مع إضاءة وظلال ودوران حول النموذج، وفتحات حقيقية في الجدران.
- **طوابق متعددة** بمناسيب وارتفاعات قابلة للتعديل.
- **المعالج السحابي** (`/api/cloud/analyze`): يحسب مساحات الجدران والأرضيات، أحجام الخرسانة، الأوزان، والتكلفة التقديرية.
- **المكتبة السحابية** (`/api/cloud/library`): أنواع جدران وأبواب ونوافذ ومواد جاهزة.
- **مدقّق النموذج** (`/api/cloud/validate`): يكتشف الأخطاء (جدران صفرية، فتحات أكبر من الجدار...).
- **ملف مشروع `.revt`**: احفظ وافتح مشاريعك (شبيه بملف مشروع Revit) — يستطيع الطلاب تبادله والعمل عليه.
- **تصدير**: `.glb` للنموذج ثلاثي الأبعاد، و`CSV` لجدول الكميات.
- **حفظ تلقائي** في المتصفح (localStorage) فلا تفقد عملك.
- **واجهة عربية RTL** ثابتة مصمّمة للهاتف (iPhone 11 Pro Max) مع دعم safe-area.

## 🧱 التقنيات

- **Next.js 14** (App Router) + **TypeScript**
- **Three.js** للعرض ثلاثي الأبعاد
- **Zustand** لإدارة الحالة
- **Tailwind CSS** للواجهة
- **Vercel Serverless Functions** للمعالج والمكتبة السحابية

## 🚀 التشغيل محليًا

```bash
npm install
npm run dev
```
ثم افتح http://localhost:3000

## ☁️ النشر على Vercel

1. ارفع المشروع إلى مستودع GitHub.
2. في Vercel: **New Project** → اختر المستودع → **Import** → **Deploy**.
3. لا تحتاج أي مفتاح أو إعداد — يعمل مباشرة. (اختياري: `NEXT_PUBLIC_CURRENCY` لعملة التكلفة).

Vercel يكتشف Next.js تلقائيًا ويثبّت الحزم ويبني المشروع.

## 📐 كيف تستخدمه (للطلاب)

1. من شريط الأدوات اختر **جدار** ثم انقر نقطتين على المسقط لرسم جدار.
2. **أرضية**: انقر ركنين لتحديد مستطيل. **عمود**: نقرة واحدة.
3. **باب/نافذة**: اختر النوع من المكتبة ثم انقر على جدار.
4. بدّل إلى **3D** لمشاهدة النموذج.
5. من تبويب **الكميات** اضغط **تحليل** لحساب الكميات سحابيًا.
6. من القائمة (☰) احفظ ملف **.revt** أو افتح ملفًا سابقًا.

يوجد مشروع جاهز للتجربة: `public/sample-house.revt` — افتحه من القائمة.

```
  store.ts                إدارة الحالة (Zustand)

  build3d.ts              بناء المشهد ثلاثي الأبعاد
  boq.ts                  حساب الكميات
  library.ts              المكتبة الافتراضية
  revt-file.ts            قراءة/كتابة ملف .revt
components/
  TopBar / ToolStrip / PlanEditor / Viewport3D /

  sample-house.revt       مشروع فيلا نموذجي
```

## 🗺️ صيغة ملف .revt

ملف JSON بسيط ومفتوح يحتوي: `levels` (الطوابق)، `materials` (المواد)، `elements` (جدران/أرضيات/أعمدة/أبواب/نوافذ). متوافق وقابل للتطوير.

## 🔮 خارطة الطريق (أفكار للتوسع)

- قص فعلي للفتحات بـ CSG، أسقف مائلة، درج، أثاث.
- تعاون لحظي عبر قاعدة بيانات سحابية (Supabase/Firebase).
- معالج سحابي لتحليل إنشائي (أحمال/عزوم) وتصدير IFC.
- مساعد ذكاء اصطناعي يولّد المخططات من وصف نصي.
